# Do Next

1. **Прочитай** `docs/current_tasks.md`

2. **Если есть задача [IN PROGRESS]:**

3. **Если нет активных задач:**

```markdown

### План
- [ ] Шаг 1
- [ ] Шаг 2
- [ ] ...

### Заметки
(пусто пока)
```

4. **Начни работу** над первым незавершённым шагом
