# Wrap Up

Выполни по порядку:

1. **Обнови `docs/current_tasks.md`:**
   - Отметь выполненные шаги `[x]`
   - Добавь новые шаги если появились
   - Запиши важные решения в "Заметки"
   - Укажи стрелкой текущий шаг: `← текущий`

2. **Сделай коммит** если есть несохранённые изменения:
   ```bash
   git add -A && git commit -m "wip: <краткое описание>"
   ```

3. **Выведи статус:**
   ```
   ## Сессия завершена

   **Сделано:** <что сделали>

   **Следующий шаг:** <что делать дальше>

   **Команда для продолжения:** `/do_next`
   ```

4. **Скажи пользователю:** "Можно закрыть сессию. Для продолжения: `/do_next`"
