# Дизайн-задачи

Что нужно заказать у дизайнера.

## Базовый шаблон

Основной шаблон для отображения контента (заметок, статей).

### Переиспользуемые компоненты

Компоненты должны быть гибкими и переноситься между темами:

- **Поиск** — модальное окно / выпадающий список
- **Избранное** — список сохранённых заметок
- **Вход/регистрация** — формы авторизации
- **Профиль пользователя** — меню, настройки
- **Навигация** — sidebar, breadcrumbs, TOC
- **Уведомления** — toast, alerts

### Страницы

- Главная (список заметок / dashboard)
- Страница заметки
- 404 / ошибки
- Онбординг (пустой сайт)

## Шаблон лендинга

Отдельный шаблон для промо-страниц.

- Hero-секция
- Блоки с фичами
- Pricing
- FAQ
- CTA-блоки

## Страницы оплаты и подписок

- Выбор тарифа / подписки
- Страница оплаты
- Успешная оплата
- Ошибка оплаты
- Управление подпиской (отмена, смена тарифа)
- Paywall (превью контента + призыв к подписке)

## Заметки

- Компоненты должны работать в разных темах (светлая/тёмная, разные стили)
- Mobile-first
- Accessibility (a11y)
