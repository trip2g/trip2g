# Emoji Bot

Простой Telegram бот для получения ID кастомных emoji.

## Как использовать

1. Создайте бота через [@BotFather](https://t.me/botfather)
2. Получите токен
3. Запустите бот:

```bash
export TELEGRAM_BOT_TOKEN="your_bot_token_here"
go run main.go
```

4. Отправьте боту сообщение с кастомным emoji
5. Бот вернёт готовый код для вставки в Obsidian:

```
![emoji](tg://emoji?id=5368324170671202286)
```

## Как работает

Бот получает сообщения через long polling, проверяет наличие `custom_emoji` entities и извлекает `custom_emoji_id`. Затем возвращает готовый markdown код в формате, который поддерживает ваш сервис публикации.

## Зависимости

Использует только стандартную библиотеку Go - никаких внешних зависимостей.
